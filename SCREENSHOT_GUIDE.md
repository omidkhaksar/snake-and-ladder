## Screenshot Settings:

### Resolution:
- **Minimum:** 1920x1080 (Full HD)
- **Recommended:** 2560x1440 (2K) or higher
- **Format:** PNG for quality, JPG for smaller file size

### Browser:
- Use **Chrome** or **Edge** for best rendering
- Enable hardware acceleration
- Clear cache before screenshots

### Lighting:
- Make sure room lighting doesn't create screen glare
- Use browser's full-screen mode (F11)
- Disable browser extensions that might show in UI
